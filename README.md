# progIII_appdev
プログラミングIIIチーム開発

回路図絵画アプリ（仮）
・手書き回路図を撮影して、pngにして書き出す。
・写真を読み込んで、書き出す。
・アプリ内で撮影（努力目標）
・素子の名前を表示する。一覧表にして表示。

○必要な機能
・画像認識
・画像書き出し

↓認識して欲しいもの（最低限）
・線
・抵抗
・直流電源
・電球

○動作環境
macOS

○言語
Python
Flask/Django/Tkinker/Kivy
OpenCV
OpenCV(contrib)
Numpy
schemdraw(?)

○主な動作
素子を割り当てたARマーカを読み込んで、回路図に変換する。


○GUI
・ファイル読み込み
・プレビュー・素子名一覧表
・ファイル書き出し
